h1. The Snippets Textmate Bundle.

h2. Installation

<pre>
mkdir -p ~/Library/Application\ Support/TextMate/Bundles
cd ~/Library/Application\ Support/TextMate/Bundles
git clone git://github.com/snippets/Snippets-tmbundle Snippets.tmbundle
</pre>
